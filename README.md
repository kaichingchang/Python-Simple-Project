# Python Simple Project

簡單 Python 的 MVC 專案，分成終端機及 Tk 兩個版本。

啟動終端機版本，請用以下指令

```
$ python3 cmd.py
請猜出 0 到 9 之間的正整數...
總共有三次機會猜出正確數字...
準備好了嗎...
請輸入：5
答對！
你猜了1次...
$
```

啟動 Tk 版本，請用以下指令

```
$ python3 controller.py

```
